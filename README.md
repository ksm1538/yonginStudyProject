# yonginStudyProject
대학교 졸업 팀 프로젝트입니다.

스터디를 하고 싶지만, 구할 방법이 없어 혼자 공부하는 사람들을 위해 간단한 가입으로 스터디를 구하고 시험 일정 공유, 채팅, 쪽지, 게시판 기능을 이용할 수 있는 웹사이트입니다.

## 개발 환경
- eclipse, Spring Framwork, Oracle, Apahce, MyBatis, Maven, Git, Java, Jsp
